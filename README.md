# Hej — Homepage

Static marketing/landing page for **Hej**, served via GitHub Pages.
Email signup is wired to Brevo (EU) with consent + confirmation state.

Brand assets (favicon, og-image, social covers + profile) are generated files —
the canonical design guide (colors, typography, logo rules, slogan) lives in
Notion: "Hej — Business" → Brand. Keep this repo's tokens in sync with it.

Main (private) repo: the Hej app + docs live separately (local sibling checkout: `../app`).
