# Heeij — SoMe-strategi & hashtag-system

Operationel guide til sociale medier. Kanon for tone, kanaler, indholdssøjler og **hashtags**.
Bruges hver gang der laves opslag (jeg poster via API — se `../.hej/README.md`).
Sidst opdateret 2026-07-09.

---

## 0. Grundpræmis (paradokset, sagt højt)
Heeij er en anti-feed-app — men vi *bruger* sociale medier til at få folk **væk** fra dem.
Det er ikke en modsætning, det er budskabet: vores SoMe er en invitation ud i virkeligheden,
ikke endnu et sted at scrolle. Hver post skal kunne ende med "…så luk appen og mød nogen".

## 1. Stemme & visuelt
- **Sprog:** dansk, varmt, ligefremt, aldrig corporate. Du/dig.
- **Visuelt:** brand-systemet — papir-baggrund, navy tekst, koral-accent, Bricolage + Caveat,
  og **opslagstavle-noterne** som gennemgående motiv (se `../brand.html`).
- **Anti-metrics:** vi hylder ikke likes/følgere. Ingen "følg for follow", ingen vanity.

## 2. ⚖️ Guardrail (gælder indtil videre)
**Ingen "skriv dig op"-CTA og intet link i opslag**, så længe dataansvarlig-blokeringen står.
Vi opbygger tilstedeværelse og brand — vi driver ikke signup-trafik og indsamler ingen data
via SoMe endnu. (Ophæves når den juridiske P0 er løst.)

## 3. Kanaler & roller
| Kanal | Rolle | Kadence (pre-launch) |
|---|---|---|
| **Instagram** (@heeijapp) | Primær: visuel + hyperlokal Storkøbenhavn | 2-3× / uge |
| **Facebook** (Heeij) | Fællesskab + events (Facebook Events når vi seeder aktiviteter) | 1-2× / uge |
| **LinkedIn** (Heeij) | Founder-historie, mission/impact, venue-partnere, rekruttering | 1× / uge |
| TikTok/Threads (senere) | Reserveret @heeijapp, aktiveres ved behov | — |

## 4. Indholdssøjler (rotér mellem dem)
1. **Aktivitets-inspiration** — "Hvem vil med?"-noter (kaffe, løb, koncert, tur). Signaturformatet.
2. **Mission / mod ensomhed** — hvorfor Heeij findes (1 ud af 6 unge er svært ensomme). Brug med omtanke.
3. **Sådan virker det & tryghed** — vært godkender, offentligt rum, ingen scroll.
4. **Storkøbenhavn** — kvarterer, steder, "det bedste ved at mødes i byen".
5. **Bag om / rejsen** — build-in-public, optælling mod launch, founder.
6. **Brugerhistorier** (efter launch) — "vi mødtes på Heeij" (UGC, se kampagne-tag).

## 5. HASHTAG-SYSTEM

**Instagram-opskrift: 5-8 tags pr. opslag** = Kerne (3) + Geo (1-2) + Tema (1-2) + evt. Aktivitet (0-1).
Kvalitet > kvantitet — små, målrettede danske tags slår store engelske (drukner ikke i #friends).

### 🔒 Kerne — GENGANGERE (altid, på hvert IG-opslag)
```
#Heeij  #NyeVenskaber  #Storkøbenhavn
```
`#Heeij` bygger brandet · `#NyeVenskaber` er missionen · `#Storkøbenhavn` er markedet.

### 📍 Geo-pulje (vælg 1-2, match postens sted)
```
#København  #Kbh  #Frederiksberg  #Lyngby  #Gentofte
#Nørrebro  #Vesterbro  #Østerbro  #Amager
```

### 💛 Tema-pulje (vælg 1-2)
```
#MødIVirkeligheden  #Fællesskab  #MødNyeMennesker
#VennerIKøbenhavn  #DetVirkeligeLiv  #LukAppenMødVerden
```

### 🎟️ Aktivitets-pulje (vælg 0-1, match indhold)
```
#TingAtLaveiKbh  #WeekendIKøbenhavn  #KulturKbh  #KaffeKbh  #UdeIDetFri
```

### 🧡 Kampagne / UGC-tag (brugerhistorier — efter launch)
```
#MødtPåHeeij
```
Vores ejede tag: opfordr folk til at bruge det, når de mødes gennem Heeij. Byg et arkiv af ægte møder.

### ⚠️ Bruges med omtanke (kun på ægte mission-/awareness-indhold, aldrig som reach-hack)
```
#Ensomhed  #EnsomhedBlandtUnge
```

